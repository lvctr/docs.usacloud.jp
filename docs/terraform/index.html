Terraform for さくらのクラウド(v2)

概要

Terraform for さくらのクラウドとは、 Terraformからさくらのクラウドを操作するためのTerraform用プラグインです。

Note

このドキュメントはterraform-provider-sakuracloud v2を対象としています。
v1をご利用の場合は以下のドキュメントを参照してください。

Terraform for さくらのクラウド(v1)

Note

このドキュメントはterraform-provider-sakuracloudの英語版ドキュメントを元に翻訳/加筆したものです。
最新のドキュメントはterraform-provider-sakuracloudのリポジトリのwebsiteディレクトリを参照してください。

https://github.com/sacloud/terraform-provider-sakuracloud/
